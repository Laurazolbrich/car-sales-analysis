# car-sales-analysis

This project is a simple Streamlit web application that allows users to explore a dataset of used car sales.

## Features
- Display a histogram of car prices
- Display a scatter plot comparing odometer readings and prices
- Interactive checkboxes to toggle visualizations
- Clean and intuitive interface for basic exploratory data analysis

## How to run the app
1. Activate your virtual environment
2. Run the command: streamlit run app.py
3. The app will oepn automatically in your browser

## Dataset
The dataset used in this project is 'vehicles.csv'. which was placed inside a folder named 'Data' in the project directory.

## Tecnologies Used
- Python
- Streamlit
- Pandas
- Plotly Express
